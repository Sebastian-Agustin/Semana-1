Student enrollment system

## Qué hace
Sistema que evalúa alumnos según su edad y los acepta o rechaza para un curso de programación.

## Lo que apliqué
- Arrays de objetos
- filter para separar aceptados y rechazados
- map para transformar los datos
- forEach para mostrar mensajes personalizados
- reduce para calcular el promedio de edad

## Aprendizajes
- Composición de funciones
- Callbacks
- Métodos de arrays encadenados
- Validaciones con typeof
